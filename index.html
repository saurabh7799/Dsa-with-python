Adobe Analytics Organization Structure

Organization (Company Level Account)

This represents the company that owns the Adobe Analytics account.

Example: ABC Retail Company
▼

Org ID

A unique identifier assigned by Adobe to the company.

Used for authentication and API access.

Example: ABC123@AdobeOrg

Global Company ID

A short identifier used in API calls to access data.

Example: mycompany

API Endpoint:
https://analytics.adobe.io/api/mycompany/reports

▼

Report Suites (Data Containers / Databases)

Report suites store analytics data for a specific website, mobile app, or digital property.

rsid: company_global_site

Analytics data collected from the main company website.

📊 Dimensions (WHAT - Descriptive Attributes)

  • variables/page "/home"
  • variables/country "India"
  • variables/device "Mobile"
  • variables/browser "Chrome"
  • variables/campaign "Summer Sale"

📈 Metrics (HOW MUCH - Numerical Values)

  • metrics/visits 8,000
  • metrics/pageviews 12,000
  • metrics/revenue $25,000
  • metrics/orders 450
  • metrics/bounceRate 35%

⚡ Events (User Actions)

  • event1 Add to Cart
  • event2 Checkout
  • event3 Purchase
  • event4 Login
  • event5 Newsletter Signup

rsid: company_mobile_app

Analytics data collected from the mobile application.

📊 Dimensions (WHAT - Descriptive Attributes)

  • App Version "v2.5.1"
  • Mobile Device Type "iPhone 14"
  • Operating System "iOS 17"
  • Country "USA"

📈 Metrics (HOW MUCH - Numerical Values)

  • App Launches 15,000
  • Screen Views 45,000
  • In-App Purchases $8,500

⚡ Events (User Actions)

  • App Install New Installation
  • App Login User Authentication
  • Purchase Completed Order
  • Add to Wishlist Saved Item
